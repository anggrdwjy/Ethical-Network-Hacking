# Routersploit
source : https://github.com/threat9/routersploit

Setup :
---
ONT Router
```
python rsf.py
show exploits
search [payload] 
use exploit [payload] case router ONT
show options
set target [ip]
exploit
```
MikroTik Router
```
python3 rsf.py
show exploits
search [payload] 
use exploit [payload] case router Mikrotik
show options
set target [ip]
exploit
```
